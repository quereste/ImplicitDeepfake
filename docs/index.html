ImplicitDeepfake is a one-shot 3D deepfake generator

Abstract

We present the first model that produces a 3D deepfake.

The idea of our pipeline is symbolically represented in the picture above, although its understanding requires some prior knowledge. Recent development in the area of three dimensional rendering have resulted in emergence of several machine learning models able to retrieve a complete 3D representation of an object from a set of 2D pictures (of quite small level of cardinality - like several dozens) with associated precise positions of camera from which the photos were taken.

By complete representation we mean that after the model is trained, we can ask it about a view from any angle expressed in real numbers and receive a novel rendered picture. If you have not come across any of such works, we strongly recommend becoming familiar with e.g. 3D Gaussian Splatting for Real-Time Radiance Field Rendering and NeRF: Representing Scenes as Neural Radiance Fields for View Synthesis. We used both of them as black box tools, measuring which of them will score better in the reconstruction metrics.

At the beginning we create a training set of photos of an anonymous 3D model and save the associated camera angles from which the photos were taken. Then we use a classical 2D deepfake model on every photo from the training set, so that we receive a fully compatible input for any of the 3D representation techniques. The experiments we have conducted show that both Gaussian Splatting and NeRF create a plausible representation for such prepared dataset.

As you for sure have noticed, the approach is one-shot in terms of number of pictures of person-to-be-mimicked. The base dataset of 2D pictures of the 3D model does not need to have anything in common with the celebrity, although we achieve better results when it bears some resemblance to the target person.

Above we present an example of a 2D image transformation. On the left there is a photo taken using blender software of a 3D anonymous model (by anonymous model we mean that the model is a matter of choice). Above the arrow there is a picture from the CelebA dataset of Ms. Céline Dion. On the right we present the result of applying GHOST 2D deepfake on anonymous and CelebA photo labelled as target and source respectively.

We process all the photos in the similar manner. Afterwards, we treat them as an input to a 3D representation model to achieve the results presented below.

Results

As we already stressed, the strengh of our approach is the ability to retrieve a deepfake view of a person from every possible angle while we were given only one specific photo.

You are free to download a 3D model underneath and play around with it using one of marvelous rendering websites designed for this purpose. Note how the quality of the three dimensional model decreases when the camera view approaches from the behind - that's acceptable as we only aim to mimic the face of the subject.
